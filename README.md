# Fire Alarm IoT System using Blynk

## Overview
This project is an IoT-based fire detection system that uses sensors to detect fire and sends real-time alerts through the Blynk mobile application. The system also activates an alarm to warn users immediately when danger is detected.

## Problem Statement
Fire accidents can happen unexpectedly, so this system is designed to provide early detection and fast notification to improve safety and response time.

## How it works
- Fire/smoke sensor continuously monitors the environment
- Microcontroller processes sensor data
- If danger is detected, the system triggers an alarm (buzzer)
- A real-time alert is sent to the Blynk mobile application
- User receives instant notification for quick action

## Features
- Real-time fire detection
- Instant mobile notifications via Blynk
- Automatic alarm activation (buzzer)
- Sensor-based monitoring system
- Simple and efficient safety solution
